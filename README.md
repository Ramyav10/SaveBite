# SaveBite - Food Delivery & Food Rescue Platform

A full-stack web application built with Java, JSP, Servlets and MySQL 
that connects customers, restaurants and NGOs on a single platform.

## Features
- Multi-role system (Customer, Restaurant, NGO, Admin)
- Complete food ordering with cart and multi-payment checkout
- Real-time order tracking with 5 delivery stages
- Restaurant dashboard with menu management and file upload
- NGO Rescue Hub for surplus food donations
- Admin panel for complete platform management
- Role-based access control with Servlet Filter security

## Tech Stack
Java | JSP | Servlets | MySQL | JDBC | Apache Tomcat 9 | HTML | CSS

## Setup
1. Clone the repository
2. Import into Eclipse as Dynamic Web Project
3. Configure MySQL database using schema.sql
4. Deploy on Apache Tomcat 9
5. Access at http://localhost:8080/savebite_cag
